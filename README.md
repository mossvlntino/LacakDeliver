# 📦 LacakDeliver: Shipment Tracking

## 📖 Overview
An industrial logistics application designed to track shipments in real-time. By integrating a dynamic timeline and a Leaflet.js map, this tool simplifies the tracking process for operators, turning raw logistical data into an easy-to-read visual interface.

## 🛠️ Focus Areas
* **Data Visualization:** Mapping physical routes using OpenStreetMap APIs.
* **Form Validation:** Ensuring reliable user input for tracking IDs.
* **Process Improvement:** Streamlining the UX for factory floor operators.
